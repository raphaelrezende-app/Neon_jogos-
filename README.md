# ⚡ NeonBrawl Studios — Landing Page

Site de landing page para a empresa de jogos online **NeonBrawl Studios**, desenvolvido com HTML, CSS e JavaScript puro.

## 🚀 Como hospedar no GitHub Pages

### Passo 1 — Crie um repositório no GitHub
1. Acesse [github.com](https://github.com) e faça login
2. Clique em **"New repository"** (botão verde)
3. Dê um nome ao repositório, ex: `neonbrawl-site`
4. Deixe como **Public**
5. Clique em **"Create repository"**

### Passo 2 — Faça o upload dos arquivos
1. Na página do repositório recém-criado, clique em **"uploading an existing file"**
2. Arraste ou selecione os arquivos: `index.html` e `README.md`
3. Clique em **"Commit changes"**

### Passo 3 — Ative o GitHub Pages
1. Vá em **Settings** (aba no topo do repositório)
2. No menu lateral, clique em **Pages**
3. Em **"Branch"**, selecione `main` e a pasta `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 minuto e acesse o link gerado:  
   `https://SEU_USUARIO.github.io/neonbrawl-site/`
